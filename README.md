# CyberLens-Ai
CyberLens AI is a real-time AI-powered cybersecurity platform that detects scams, phishing, fake job offers, malicious URLs, QR fraud, and suspicious PDFs. Built with Python, Streamlit, ML, NLP, OCR, and OpenCV, it combines multilingual threat detection, ensemble AI models, and a cyberpunk-inspired dashboard to protect users from digital fraud.
